# Flask ML Classifier

This is a Flask web application that uses a machine learning model
to predict results from user input.

## Project Structure

FLASK3/
├── templates/index.html
├── app.py
├── classifier.py
├── train_reg.py
├── model.pkl
├── wsgi.py
├── requirements.txt
└── README.md

## Installation

pip install -r requirements.txt

## Run

python app.py

Open: http://127.0.0.1:5000/

## Usage

Enter input and get prediction.

## Technologies Used

- Python
- Flask
- Scikit-learn
- HTML
